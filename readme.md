## Project Description

MamaMia Meal Kits is a guacamole-themed meal kit delivery service. Users can browse weekly meal kits, register for an account, and log in as either a **customer** or a **data entry clerk**. The app features full server-side form validation, password encryption, MongoDB user storage, and automated welcome emails via Mailgun.
